# Инструкция по MarkDown

## Заголовки

Заголовки должны начинаться с новой строки и имеют 6 размеров, каждый размер обозначается от 1 (самый большой) до 6 символов <span style="color:#0fb4ff;">#</span> перед названием заголовка с пробелом после символов

## Выделение текста

<span style="color:#0fb4ff;">*курсив*</span> - \*курсив* или \_курсив_ обрамляем текст c двух сторон без пробелов одной <span style="color:#0fb4ff;">*</span> или одним <span style="color:#0fb4ff;">_</span>

<span style="color:#0fb4ff;">**полужирный**</span> - \*\*полужирный** или \_\_полужирный__ обрамляем текст c двух сторон без пробелов двумя <span style="color:#0fb4ff;">**</span> или двумя <span style="color:#0fb4ff;">__</span>

<span style="color:#0fb4ff;">***полужирный курсив***</span> - \*\*\*полужирный курсив*** или \_\_\_полужирный курсив___ обрамляем текст c двух сторон без пробелов тремя <span style="color:#0fb4ff;">***</span> или <span style="color:#0fb4ff;">___</span>

<span style="color:#0fb4ff;">_также можно **комбинировать**_</span> - \_также можно \*\*комбинировать**_

Что бы игнориировать выделение текста как написанно выше, т.е. написать символ, используемый в Markdown, поставьте перед ним <span style="color:#0fb4ff;">\​</span> обратный слэш -  \​_также можно \​*\​*комбинировать**_

<span style="color:#0fb4ff;"></span>

## Работа с изображениями

Вставить изображение в текст

!​[ ]​( ) - всё без пробелов  
в начале <span style="color:#0fb4ff;">!</span>  
в <span style="color:#0fb4ff;">[ ]</span> текст внутри можно с пробелами вместо картинки, вслучае если картинка недоступна  
 в <span style="color:#0fb4ff;">( )</span> внешняя или внутренняя ссылка на саму картинку

!\[</span>Avatar example](./images/avatar.png) - в данном случае картинка лежит в папке /images, которая расположена внутри рабочей папки с файлом
